Project MCC Library

Project ini merupakan project akhir dari program MCC yang dibuat oleh peserta MCC batch ke-47. Project ini dibuat dengan tujuan untuk memudahkan peserta MCC yang akan datang untuk melihat referensi project akhir mereka, dan juga sebagai tempat melakukan penyimpanan seluruh project peserta MCC agar dapat disimpan dengan baik dan aman.
